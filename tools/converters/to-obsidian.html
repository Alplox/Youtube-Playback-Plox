YouTube Playback Plox

Export → Obsidian Bases

Convierte tu backup de YouTube Playback Plox a Markdown compatible con Obsidian Bases.

1 - Elige el modo de exportación
Modo ZIP - Obsidian Bases
Genera un .zip con una nota .md por video dentro de una carpeta YouTube/. Al descomprimirlo en tu vault y crear una Base apuntando a esa carpeta, podrás ver tus videos como tarjetas con propiedades filtrables (channel, percentWatched, type, etc.).
2 - Sube tu archivo de backup
Arrastra el archivo aquí o haz clic para buscar
Formatos aceptados:
JSON nativo (.json) FreeTube (.db / .ndjson)
3 - Configura la salida
Incluir embed de thumbnail ![thumbnail](url)
Solo videos completados o casi completados (≥ 85%)
Usar callout blocks de Obsidian para metadatos
Incluir fecha como prefijo en nombre de archivo (modo ZIP)
